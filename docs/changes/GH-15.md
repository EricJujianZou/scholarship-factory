# GH-15 — Seed-list model + TOML loader

Adds `scholarship_factory/seeds.py`: `Seed` (pydantic v2) + `SeedType` enum + a
`load_seeds` TOML loader (stdlib `tomllib`) — Session 3 / Fetch groundwork.

- `SeedType` represents all source types incl. the auth-walled `instagram`/`x`
  (deferred to their own session); adapters later decide which they fetch.
- `Seed`: `type`, `value`, `enabled=True`, optional `label`. Unknown `type`
  rejected.
- `load_seeds` parses `[[seeds]]` tables into `list[Seed]`; empty file -> `[]`;
  malformed entry -> `ValidationError`.
- No fetching, no adapters, no network.

Tests: `tests/test_seeds.py`. Built manually (harness stalled; see engine #92/#93).
