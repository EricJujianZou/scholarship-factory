# GH-14 — Applicant profile model + store

Adds `scholarship_factory/profile.py`: `ApplicantProfile` (pydantic v2) + a
SQLite `ProfileStore`, the S6 foundation ranking/filtering will match against.

- v1 shape (minimal + tags): `region`, `education_level`, `field_of_study`,
  `tags: list[str]`, `bio` (free text), plus `owner="me"` and timestamps.
- Store insert/get/list/update; `tags` serializes to JSON text and round-trips;
  configurable db path; `update` refreshes `updated_at` only.
- No ranking/matching/opportunity coupling (later ticket).

Tests: `tests/test_profile.py`. Built manually (harness stalled; see engine #92/#93).
